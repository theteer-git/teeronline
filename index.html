Shillong Teer FR and SR schedule

The declared First Round time is 4:15 PM and the declared Second Round time is 5:15 PM IST. Actual publication may occur slightly before or after the declared time. The regular Shillong day game is generally not scheduled on Sunday.

How to read the live result status

FR means First Round and SR means Second Round. Each published result is displayed as a two-digit number from 00 to 99. A pending label means that the relevant round has not yet been recorded for the date shown; it does not mean the result is zero. Always check the displayed date before using a number, particularly after returning to a browser tab that has remained open for a long time.

TeerOnline records result information for public reference and updates the live display when a valid result is available.
